## 📌 About PDX 2026

**Paradox World** is an interactive web project featuring stories, games, science content, and mysteries. A creative space where imagination meets technology.

---

## ⚡ Features

- 📖 **Stories** - Original creative writing & fiction
- 🎮 **Games** - Interactive browser-based games
- 🔬 **Science** - Fascinating science facts & experiments
- 🔮 **Mystery** - Unsolved mysteries & conspiracies
- 🌐 **Interactive UI** - Smooth, modern web experience

## 🚀 Local Setup

```bash
git clone https://github.com/sachin-null/pdx-2026.git
cd pdx-2026

# Open in browser
open index.html
# or
python -m http.server 8080
```

Then visit: `http://localhost:8080`
